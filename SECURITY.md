# Політика Безпеки (Security Policy)

Дякуємо, що дбаєте про безпеку розширення **Temu Smart Score & Bar (UA) — Pro**.

## Підтримувані Версії

Ми надаємо оновлення безпеки лише для останньої мажорної версії розширення.

| Версія | Підтримується? |
| --- | --- |
| 15.x | :white_check_mark: |
| < 15.0 | :x: |

## Як повідомити про вразливість

Якщо ви знайшли вразливість у безпеці (наприклад, витік даних через XSS у вікні iframe, слабкість в IPC комунікації), **будь ласка, НЕ створюйте публічний Issue**.

Замість цього, надішліть приватне повідомлення або створіть Security Advisory через інструменти GitHub у вкладці **Security**. Ми намагаємось швидко реагувати на всі сповіщення та випускати патчі якомога швидше, оскільки безпека користувачів у браузері є нашим найвищим пріоритетом (особливо при роботі з e-commerce сайтами).

## Принципи Безпеки Розширення
1. Розширення не збирає та не відправляє жодних персональних даних, паролів чи історії платежів.
2. Розширення працює виключно на домені `*://*.temu.com/*`.
3. Використовується сувора політика CSP у межах концепції Manifest V3 (відсутність `eval`, виключно безпечні API запити).
