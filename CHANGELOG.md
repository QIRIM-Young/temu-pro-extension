# Журнал змін (Changelog)

Усі помітні зміни у цьому проєкті будуть задокументовані в цьому файлі.

Формат базується на [Keep a Changelog](https://keepachangelog.com/uk/1.0.0/), і цей проєкт дотримується принципів [Semantic Versioning](https://semver.org/lang/uk/).

## [15.4.0] - Поточна версія
### Додано
- 🆕 Новий користувацький інтерфейс (`sidepanel`) з кастомним ефектом "Скла" (Glassmorphism).
- 🆕 Інтерактивні повзунки для налаштування ваги критеріїв Smart Score (Знижка, Рейтинг, Продажі, Відгуки).
- 🆕 Автоматичний конвертер валют ExchangeRate-API для адаптації цін у UAH.
- 🆕 Інтернаціоналізація бейджів та підказок українською мовою.
- 🆕 Додано Manifest V3 Service Worker для фонових операцій (без background pages).
- 🆕 Додано інструкції для контриб'юторів, шаблони GitHub Issues та Pull Requests.

### Виправлено
- 🐛 Злам CSS-верстки сторінок Temu. Тепер панель розширення живе у власному ізольованому `iframe`.
- 🐛 Проблема з перекриттям прозорості (backdrop-filter) між `iframe` і `wrapper` під час відображення панелі.
- 🐛 Баг, через який не вдавалось повторно увімкнути панель після закриття хрестиком (X).

### Змінено
- 🔄 Оновлено архітектуру `content.js`, тепер він використовує надійний IPC (`window.postMessage`) для взаємодії з UI.
